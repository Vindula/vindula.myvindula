vindula.myvindula Installation
------------------------------

To install vindula.myvindula using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``vindula.myvindula`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        vindula.myvindula
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
